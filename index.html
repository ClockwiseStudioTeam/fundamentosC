1 Fevereiro: Aprenda o que é a programação e a sua utilidade

A programação é a arte e ciência de escrever código para computadores para realizar tarefas específicas. Os programadores usam uma variedade de linguagens de programação, como Python, JavaScript, C, C++ e Java, para escrever códigos que podem ser executados pelos computadores. A programação é uma habilidade valiosa em uma variedade de campos, desde tecnologia da informação e ciência da computação até finanças e engenharia. Aprender a programar pode ser desafiador no início, mas é uma habilidade gratificante e valiosa ao longo do tempo.

1 Fevereiro: O que é um IDE?

IDE (ambiente de desenvolvimento integrado) é uma ferramenta de programação que fornece uma interface para escrever, depurar e executar códigos. O Visual Studio Community é uma das IDEs mais populares e é desenvolvido pela Microsoft. Ele é gratuito para uso pessoal e de pequenas equipes e suporta uma variedade de linguagens de programação, incluindo C, C#, C++, Python e JavaScript. Ele também tem recursos avançados como integração com outras ferramentas da Microsoft, como o Azure e o TFS, e oferece suporte para desenvolvimento de aplicativos para Windows, iOS, Android e web. Em resumo, o Visual Studio Community é uma excelente opção para desenvolvedores iniciantes ou experientes que procuram uma IDE robusta e completa.

7 Feveiro: Texto-Resumo: Estruturas de Controlo e operadores lógicos: if ,if-else, if-else if-else

TAREFA: Apenas ler e experimentar

Estrutura if-else: A estrutura if-else é usada para tomar decisões em um programa. Ela verifica se uma determinada condição é verdadeira ou falsa. Se a condição for verdadeira, o código dentro do bloco if é executado. Se a condição for falsa, o código dentro do bloco else é executado. 

          int idade = 18;
          
          if (idade >= 18) {
              printf("Você é maior de idade");
          } else {
              printf("Você é menor de idade");
          }

Estrutura if-elseif: A estrutura if-elseif é usada quando há mais de duas opções possíveis e somente uma delas deve ser escolhida. Ela funciona como uma cadeia de verificações, verificando cada condição uma após a outra até que uma condição seja verdadeira. Quando uma condição é verdadeira, o código dentro do bloco correspondente é executado e as verificações adicionais são interrompidas. 

        int nota = 85;
          
        if (nota >= 90) {
            printf("Conceito A");
        } else if (nota >= 80) {
            printf("Conceito B");
        } else if (nota >= 70) {
            printf("Conceito C");
        } else {
            printf("Conceito D");
        }

Estrutura if-else if-else: A estrutura if-else if-else é semelhante ao if-elseif, mas ela sempre tem um bloco else no final. Isso garante que, independentemente das condições verificadas, sempre haja um bloco de código que será executado 

            int idade = 15;
            int dinheiro = 50;
            
            if (idade >= 18 && dinheiro >= 100) {
                printf("Você pode entrar na festa");
            } else if (idade >= 18) {
                printf("Você precisa de mais dinheiro para entrar na festa");
            } else {
                printf("Você é menor de idade e não pode entrar na festa");
            }

Em resumo, as estruturas if-else, if-elseif e if-else if-else são utilizadas para tomar decisões baseadas em condições, e são uma parte fundamental da programação estruturada. Isso permite ao programa tomar decisões e agir de acordo com as condições especificadas.

10 Feveiro: Assinaturas e Funções com estruturas de controlo

TAREFA: cópia do código efetuado na visualização dos videos + apontamentos em comentário

Em C, a assinatura de uma função inclui o seu nome, o tipo de retorno e os tipos de argumentos que a função aceita. Por exemplo: 

int soma(int x, int y)
Esta assinatura declara uma função chamada soma que retorna um int e aceita dois argumentos inteiros x e y.Elas permitem que executemos funções que estão situadas abaixo da função main, e que assim o compilador "saiba" da sua existência. Quanto às funções com estruturas de controle, como if, else dentro delas, isto é comum e permitido em C. As estruturas de controle permitem que a função execute diferentes blocos de código, dependendo das condições especificadas. Por exemplo: 
          int retorna_maior(int x, int y) {
            if (x > y) {
                return x;
            } else {
                return y;
            }
        }
Esta função retorna o maior dos dois argumentos inteiros x e y, dependendo da comparação feita pelo if. O return dentro das estruturas de controle indica o valor a ser retornado pela função.

13 Fevereiro:Pedir Input ao utilizador- Scanf

TAREFA: cópia do código efetuado na visualização dos videos + apontamentos em comentário

O scanf é uma função em C que permite a leitura de dados do teclado. Aqui estão alguns exemplos de uso: Não esquecer de adiciona o include do stdio.h! 


          
              int main() {
                  int x;
                  float y;
                  char z;
              
                  printf("Insira um inteiro: ");
                  scanf("%d", &x);
              
                  printf("Insira um ponto flutuante: ");
                  scanf("%f", &y);
              
                  printf("Insira um caractere: ");
                  scanf(" %c", &z);
              
                  printf("Você digitou: %d, %.2f, %c\n", x, y, z);
              
                  return 0;
              }
No exemplo acima, o scanf é usado para ler três tipos diferentes de dados: um inteiro, um ponto flutuante e um caractere. O primeiro argumento do scanf especifica o formato dos dados a serem lidos, e o segundo argumento é o endereço da variável na qual os dados serão armazenados. Note que o & é usado antes das variáveis, pois o scanf precisa do endereço da memória onde armazenar os dados lidos.

13 Fevereiro:Incrementos e For Loop(cliclo)

TAREFA: cópia do código efetuado na visualização dos videos + apontamentos em comentário

Em programação C, o operador de incremento ++ é usado para adicionar 1 a uma variável. Por exemplo: 

int x = 10;
  x++; // x agora é igual a 11
Já o operador de decremento -- é usado para subtrair 1 da variável. Por exemplo: 
int y = 10;
  y--; // y agora é igual a 9
For Loop em C: O for loop é uma estrutura de controle de repetição em C que permite executar um bloco de código várias vezes, com base em uma condição de parada. A sintaxe básica é a seguinte: 
for (inicialização; condição; atualização) {
  // bloco de código a ser repetido
}
A inicialização é realizada uma vez antes do loop ser executado pela primeira vez. A condição é avaliada antes de cada iteração e, se for verdadeira, o loop continua. A atualização é realizada após cada iteração. Quando a condição for falsa, o loop é interrompido e o controle é transferido para a linha de código após o loop.

14 Fevereiro:While e Do While (ciclos)

TAREFA: cópia do código efetuado na visualização dos videos + apontamentos em comentário

Um "do-while" loop é uma estrutura de repetição que executa um bloco de código pelo menos uma vez antes de verificar a condição de saída. O código é executado primeiro, depois a condição é verificada e, se for verdadeira, o código será executado novamente. Isso continua até que a condição seja falsa. A sintaxe para um loop "do-while" é: 

 do {
         // código a ser executado
        } while (condição);
Já o "while" loop é uma estrutura de repetição que verifica uma condição antes de executar o código. Se a condição for verdadeira, o código será executado e a condição será verificada novamente. Isso continua até que a condição seja falsa. A sintaxe para um loop "while" é: 
 while (condição) {
            // código a ser executado
          }

16 Fevereiro:Arrays

Os arrays são uma das estruturas de dados mais comuns em C. Eles permitem armazenar uma coleção de elementos do mesmo tipo, como inteiros, caracteres ou ponto flutuante, em uma única variável. Cada elemento do array é acessado por meio de um índice, que representa sua posição dentro do array. O índice começa em zero e vai até o tamanho do array menos um. Ao declarar um array, você deve especificar o tipo de elementos que ele armazenará e seu tamanho, que é o número de elementos que ele conterá. Por exemplo, para declarar um array de 10 inteiros, você escreveria: int numeros[10];. Depois de declarado, você pode atribuir valores aos elementos do array e acessá-los por meio de seus índices. Arrays são úteis para representar coleções de dados que precisam ser armazenados e processados de forma contínua.

    • 20 Fevereiro:Structs

    As estruturas (structs) são uma das formas de organizar dados em C. Elas permitem agrupar diferentes tipos de dados em uma única variável. Por exemplo, você pode criar uma estrutura para representar uma pessoa, que teria nome, idade e endereço como informações associadas. Para criar uma estrutura, você usa a palavra-chave struct seguida pelo nome da estrutura e pelos campos que ela deve conter, delimitados por chaves. Cada campo é especificado por seu tipo e nome. Por exemplo: struct pessoa { char nome[50]; int idade; char endereco[100]; };. Depois de criada, você pode declarar variáveis da estrutura e acessar seus campos por meio de um ponto (.). As estruturas são úteis para organizar dados complexos e relacionados, como registros de clientes, informações de produtos, entre outros.
